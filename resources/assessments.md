# Assessments (Rules & Guidelines)

## Individual (70%)

| Component | Weight | Details |
|-----------|--------|---------|
| Class Performance (Tutorials) | 10% | T2–T8: 1 pt each (7 pts total) + 3 bonus pts for submitting all tutorials/activities on time |
| 6 Weekly Quizzes (Canvas) | 10% | Timed; approximately every 2 sessions; **NOT available after Canvas deadline** |
| Final Exam | 50% | Open book (lecture slides provided within Examplify); No internet; Full lockdown; MCQs + Short-answer questions; In-person written |

> Tutorial late submissions: 1 day late → 50% score; any later → 0 score

## Group (30%)

| Component | Weight | Format | Deadline |
|-----------|--------|--------|----------|
| Proposal | 10% | 2 slides (cover + references not counted); 1 submission per team on Canvas | Week 3 (28 May 2026) |
| Presentation | 20% | Up to 8 slides (cover + references not counted); 10 mins + 5 mins Q&A; online via Zoom | Weeks 7–8; slides due **21 Jun 2026 (Sun midnight)** |

## Project Details

### Proposal
- 2-slide format (S1: problem context / As-Is; S2: proposed solution / To-Be)
- Cover slide and references slide do not count toward the 2-slide limit

### Presentation
- Required visuals: process diagrams, swimlanes, flowcharts
- Must show: As-Is process, user journey & pain points, To-Be process, system architecture, clear traceability
- Conducted online via Zoom in Weeks 7 or 8
